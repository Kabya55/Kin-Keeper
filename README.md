# 🤝 Kin Keeper: Friendship Analytics & Tracker

**Kin Keeper** is a modern web application that helps you stay in touch with your friends and track your social life. It keeps track of your last communication and provides the analytics you need to improve your relationships.

---

## 🚀 Technologies Used

This project was built using the latest web technologies:

- **React.js (Vite)**
- **Tailwind CSS**
- **DaisyUI**
- **Recharts**
- **Lucide React & React Icons**
- **React Toastify / Hot Toast**
- **React Router Dom**
- **Context API**

---

## ✨ Key Features

### 1. 📊 Friendship Analytics (Stats Page)

A powerful dashboard showing the ratio of calls, texts, and video calls using **Recharts**, allowing you to easily understand your social engagement.

### 2. ⚡ Quick Check-In & Real-time Feedback

Check-in can be done by clicking the Call, Text or Video button directly from each friend's profile. Each time a **Toast Notification** is clicked, the user is assured that the data has been saved.

### 3. 📜 Smart Timeline with Filtering

The Timeline page neatly organizes all your interactions by date. A **Dropdown Filter** has been added so you can find history by category.
